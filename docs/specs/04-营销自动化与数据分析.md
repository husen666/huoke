# 营销自动化 & 数据分析 - 详细功能规格

## 1. 企业微信营销

### 1.1 客户触达

#### 好友管理
| 功能 | 说明 | 频率限制 |
|------|------|---------|
| 批量添加好友 | 通过手机号/微信号批量添加 | 每日上限 30 人/员工 |
| 渠道活码 | 一码多人轮询，扫码自动分配 | 无限制 |
| 群活码 | 群满自动建新群 | 每群 200/500 人 |
| 好友标签 | 自动/手动打标签 | 无限制 |

#### 消息群发
| 消息类型 | 说明 | 频率限制 |
|---------|------|---------|
| 文字消息 | 纯文本，支持变量插入 | 每天1次/客户 |
| 图文消息 | 图文链接卡片 | 每天1次/客户 |
| 图片消息 | 海报/产品图 | 每天1次/客户 |
| 小程序卡片 | 小程序跳转卡片 | 每天1次/客户 |
| 视频消息 | 短视频内容 | 每天1次/客户 |


## 3. 营销自动化引擎

### 3.1 可视化工作流设计器

#### 节点类型

**触发节点（入口）**
| 节点 | 说明 | 配置项 |
|------|------|-------|
| 新线索进入 | 新线索创建时触发 | 来源筛选、评分筛选 |
| 标签变更 | 客户标签添加/移除时 | 目标标签 |
| 阶段变更 | 客户阶段变化时 | 目标阶段 |
| 定时触发 | 按计划执行 | Cron 表达式 |
| 日期触发 | 客户字段日期到达时 | 生日、合同到期 |
| 行为触发 | 客户执行特定行为时 | 访问页面、点击链接 |
| 手动触发 | 人工启动 | 选择目标客户 |
| Webhook | 外部系统调用 | URL、认证 |

**条件节点（判断）**
| 节点 | 说明 |
|------|------|
| 条件分支 | IF/ELSE 条件判断 |
| 多路分支 | 多条件分支（Switch） |
| A/B 分流 | 按比例随机分流 |
| 等待条件 | 等待特定条件满足 |

**动作节点（执行）**
| 节点 | 说明 |
|------|------|
| 发送企微消息 | 发送企业微信消息 |
| 发送钉钉消息 | 发送钉钉消息 |
| 发送短信 | 发送短信 |
| 发送邮件 | 发送邮件 |
| 打标签 | 添加/移除客户标签 |
| 变更阶段 | 更新客户生命周期阶段 |
| 分配负责人 | 分配/转移客户负责人 |
| 创建工单 | 自动创建工单 |
| 创建任务 | 创建跟进任务提醒 |
| 更新字段 | 更新客户字段值 |
| AI 处理 | 调用 DeepSeek 进行分析/生成 |
| HTTP 请求 | 调用外部 API |
| 延时等待 | 等待指定时间 |
| 通知 | 内部通知（钉钉/企微/站内） |

**控制节点**
| 节点 | 说明 |
|------|------|
| 并行执行 | 多个分支同时执行 |
| 合并 | 多分支合并 |
| 循环 | 重复执行（最多N次） |
| 结束 | 流程终止 |

```
  │   ├── A组 (50%): "Hi {姓名}，很高兴认识你..."

  B组: 回复率 31%, 转化率 7.8%  ← 胜出
  置信度: 95%
  建议: 采用B组方案
```

## 5. 通知系统

### 5.1 通知类型

| 类型 | 触发场景 | 推送渠道 |
|------|---------|---------|
| 新线索通知 | 有新线索进入 | 企微/钉钉/站内 |
| 客户动态 | 客户重要行为 | 企微/钉钉/站内 |
| SLA 预警 | 即将超时 | 企微/钉钉/短信 |
| 流失预警 | 客户可能流失 | 企微/钉钉/站内 |
| 工作流通知 | 自动化流程事件 | 企微/钉钉/站内 |
| 数据报告 | 定时报告生成 | 企微/钉钉/邮件 |
| 系统告警 | 系统异常 | 钉钉/短信/邮件 |

### 5.2 通知偏好设置

用户可自定义：
- 接收哪些类型的通知
- 通过哪个渠道接收
- 免打扰时段
- 通知频率（实时/汇总）
