# UI/UX 页面设计规格

## 1. 设计系统

### 1.1 设计原则

- **效率优先**：减少操作步骤，让营销人员专注核心工作
- **信息密度**：合理展示密集数据，减少翻页和跳转
- **一致性**：统一的交互模式和视觉语言
- **智能化**：AI 辅助减少重复工作
- **响应式**：桌面端优先，预留移动端适配

### 1.2 品牌色彩

```
主色 (Primary):     #2563EB (蓝色 - 专业/信任)
次色 (Secondary):   #7C3AED (紫色 - 智能/科技)
强调色 (Accent):    #F59E0B (金色 - 价值/提醒)

警告色 (Warning):   #F59E0B (橙色)
危险色 (Danger):    #EF4444 (红色)

```

#### 消息输入框功能
- **AI 按钮**：一键生成 AI 建议回复

- **Shift+Enter**：换行
- **Enter**：发送
- **@**：提及同事协助

| 筛选 | Popover 下拉，支持多选，显示已选标记 |
| 搜索 | 即时搜索(300ms防抖)，高亮匹配 |
| 分页 | 默认20条/页，支持切换 |
| 加载 | Skeleton 骨架屏 |
| 空状态 | 插画 + 引导文案 + 操作按钮 |
| 删除确认 | 二次确认弹窗，高危操作需输入确认 |
| Toast | 右上角，3秒自动消失 |
