# CRM 客户管理 - 详细功能规格

## 1. 模块概述

CRM 客户管理模块是火客系统的核心数据枢纽，承载客户全生命周期数据管理，提供 360° 客户视图，驱动精准营销决策。

---

## 2. 客户档案管理

### 2.1 客户类型

|------|------|------|------|
| 姓名 | string | 是 | 客户姓名 |

| 微信号 | string | 否 | 企微/个微 |

### 3.2 自动打标规则

```yaml
规则示例：
  - name: "高意向客户"
    conditions:
      - field: lead_score
        operator: ">="
        value: 80
      - field: last_activity
        operator: "within"
        value: "7d"
    actions:
      - add_tag: "高意向"
      - add_tag: "VIP"
      - notify: owner

  - name: "沉默客户预警"
    conditions:
      - field: last_activity
        operator: "older_than"
        value: "30d"
      - field: stage
        operator: "not_in"
        value: ["已成交", "已流失"]
    actions:
      - add_tag: "沉默预警"
      - send_notification: "客户30天未互动"
```

## 4. 客户生命周期管理

### 4.1 阶段定义

```
┌────────┐   ┌────────┐   ┌────────┐   ┌────────┐   ┌────────┐
│  潜在   │──▶│  意向   │──▶│  商机   │──▶│  成交   │──▶│  复购   │
│  客户   │   │  客户   │   │  客户   │   │  客户   │   │  客户   │
└────────┘   └────────┘   └────────┘   └────────┘   └────────┘
     │            │            │            │            │
   培育期       跟进期       谈判期       服务期       忠诚期
     │            │            │            │            │
  自动培育     销售跟进     商务谈判     交付服务     增购/续费
  内容触达     需求挖掘     报价方案     客户成功     忠诚维护
```

### 4.2 阶段转换规则

| 从 | 到 | 触发条件 | 自动动作 |
|---|---|---------|---------|
| 潜在 → 意向 | 客户主动咨询/回复消息 | 更新阶段，通知负责人 |
| 意向 → 商机 | 明确需求 + 预算 + 时间线 | 创建商机记录，排入销售管线 |
| 商机 → 成交 | 签约/付款 | 创建订单，发送感谢信 |
| 成交 → 复购 | 二次购买 | 升级 VIP，增加积分 |
| 任意 → 流失 | 90天无互动 + 明确拒绝 | 标记流失原因，进入唤醒池 |

### 4.3 流失预警

```
预警级别：
  🟡 轻度：15天未互动 → 自动推送内容
  🟠 中度：30天未互动 → 通知负责人 + AI 生成唤醒话术
  🔴 重度：60天未互动 → 升级到主管 + 电话跟进建议
  ⚫ 流失：90天未互动 + 无法联系 → 标记流失，进入唤醒池
```

## 5. 客户评分系统 (Lead Scoring)

### 5.1 评分模型

#### 规则评分（基础）
```
总分 = 行为分(40%) + 属性分(30%) + 互动分(20%) + 时效分(10%)

行为评分 (0-100):
  访问官网          +5
  查看产品页        +8
  下载白皮书/资料    +12
  注册/提交表单      +15
  参加线上活动      +18
  申请试用/演示      +25
  主动咨询客服      +20

属性评分 (0-100):
  目标行业          +25
  决策者/影响者角色   +20
  公司规模匹配      +15
  地域匹配          +10
  预算匹配          +30

互动评分 (0-100):
  打开邮件          +5
  点击链接          +8
  回复消息          +15
  参与社群讨论      +10
  转发/分享内容      +12

时效评分 (0-100):
  7天内活跃         +30
  7-14天活跃        +20
  14-30天活跃       +10
  30天以上未活跃     -20
```

#### AI 评分（增强）
```
DeepSeek 智能评分：
  输入：客户全部行为数据 + 历史对话 + 画像信息
  输出：
    - 综合意向分 (0-100)
    - 预测转化概率 (0-100%)
    - 最佳跟进时机建议
    - 推荐跟进策略
    - 评分理由说明
```

### 5.2 评分等级

| 等级 | 分数区间 | 标记 | 处理策略 |
|------|---------|------|---------|
| 🔥 极高 | 90-100 | Hot Lead | 立即跟进，优先级最高 |
| 🟠 高 | 70-89 | Warm Lead | 24小时内跟进 |
| 🟡 中 | 50-69 | Nurturing | 进入培育流程 |
| 🟢 低 | 30-49 | Cold Lead | 定期内容触达 |
| ⚪ 极低 | 0-29 | Inactive | 长期培育池 |

## 6. 客户分群

### 6.1 规则分群

可视化条件构建器，支持 AND/OR 组合：

```
示例分群：【高价值企业客户-华东地区】
  条件组 1 (AND):
    - 客户类型 = 企业
    - 地区 IN (上海, 浙江, 江苏, 安徽)
    - 客户评分 >= 70
  条件组 2 (OR):
    - 累计消费 >= 50000
    - 客户阶段 IN (成交客户, 复购客户)
```

### 6.2 AI 智能分群

DeepSeek 自动分析客户数据，发现隐含分群：
- **基于行为相似度**：聚类分析找出相似行为模式
- **基于价值预测**：预测高价值客户群体
- **基于流失风险**：识别高流失风险客户群
- **基于需求匹配**：按产品/服务匹配度分群

### 6.3 动态分群

分群条件实时计算，客户自动进出分群：
```
客户行为变化 → 重新计算评分 → 检查所有分群条件 → 更新分群归属 → 触发对应营销动作
```
